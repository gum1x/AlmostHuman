    Tunable Dials (only things you can actually change)

    Editable dials: Gate Threshold (simple pre-filter: if score below this we hard-skip the LLM entirely) and Willingness to Respond (test bias appended to the slim PRE-COMPUTED SIGNALS the 3Q model sees). Detailed per-factor Gate Weights (velocity, emotional, fatigue, feedback) are no longer shown here — we don't expose them because the qualitative decision model (the three questions) does not listen to or consume the internal gate factors at all. The gate is now just a cheap structural pre-filter; directs/continuations always force proceed to the reasoning AI. What the model actually attends to: the slim activity numbers (responses_last_hour, time_since_last_bot_msg_min, is_reply_to_bot, direct_mention) + optional willingness + current_posture + the injected persona memory + relevant context. Gate still runs and is reported in the engagement_gate step for observability.
